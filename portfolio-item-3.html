MyCrypto Dashboard

#fetchAPI #HTML #JS #bootstrap

This web app displays top 50 cryptos based on market cap with trading view charts for BUY/SELL prediction.



Features

1. list view and grid view of top 50 crypto coins with real time data from https://www.coingecko.com/

2. list view has coin name, logo, symbol, current price, current market cap, current total volume and price change percentage in 24hrs

3. grid view coin logo, symbol, current price and price change percentage in 24hrs

4. button to toggle between grid and list layout

5. button to move to hearder section from footer section

6. clicking on the coin symbol, takes user to trading view charts & widget section (charts from https://in.tradingview.com/)
